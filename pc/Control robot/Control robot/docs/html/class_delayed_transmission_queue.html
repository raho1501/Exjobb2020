Detailed Description

DelayedTransmissionQueue class.

a queue that will not dequeue a message and send it to the network unless a set time is passed.

bool DelayedTransmissionQueue::transmitt ( )

transmitt function that sends the message at the front of the queue, returns if transmission was performed or not

Function performs a check if a set time has passed. If true then it will send the first message and return true. If the set time has not passed then no transmisson of message is performed and a false is returned. Check is done by private function.
